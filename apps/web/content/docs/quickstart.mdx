Get up and running with the `apidemic` CLI runner to test your endpoints locally.

## Prerequisites

Ensure you have your target API running locally or accessible in a test environment. You will need:
- Node.js >= 18.x
- An OpenAPI / Swagger spec URL or JSON file (Optional, but accelerates initial agent discovery)

## Installation

Install the runner globally or run it directly using your preferred package manager:

```bash
pnpm dlx apidemic init
