# apidemic

apidemic is an automated agentic API testing platform. It uses an agent-driven approach to explore, validate, and stress test API endpoints eliminating the manual overhead of writing and maintaining test cases.

## What it does

Instead of static test suites, apidemic sends an autonomous agent against your API. The agent handles endpoint discovery, request generation, edge case detection, and failure reporting on its own.
